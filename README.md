# 인도 중고차 가격 예측 프로젝트
- POS_Cars(주)는 인도 중고차 시장에서 신규 사업을 목표로 하는 신생 스타트업.
- 인도 중고차 시장의 고객들은 가성비가 좋은 차량을 선호.
- POS_Cars(주)는 경쟁력 확보와 수익성 향상을 위해 데이터 분석을 통하여 "핵심영향인자 도출"과 "가격예측모델 개발"을 진행하려 함.
